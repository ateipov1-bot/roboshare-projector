Русский

1. Общие положения

Приложение «RoboShare Projector» (далее — Приложение) разработано компанией RoboSpark. Настоящая Политика конфиденциальности описывает, какие данные собираются и как они используются.

2. Какие данные мы собираем

  • Сетевая информация — название Wi-Fi сети для подключения к серверу.
  • Данные Bluetooth — для приёма презентаций с устройства учителя.
  • Данные о местоположении — только для определения названия Wi-Fi сети (требование Android 10+). Данные геолокации не сохраняются и не передаются.

3. Как мы используем данные

Собранные данные используются исключительно для:

  • Подключения к серверу RoboSpark для получения расписания и учебных материалов.
  • Приёма и отображения презентаций через Bluetooth.

4. Передача данных третьим лицам

Мы не передаём персональные данные третьим лицам. Данные передаются только на сервер RoboSpark для функционирования приложения.

5. Хранение данных

Приложение хранит кэш учебных материалов локально на устройстве. Персональные данные пользователей не сохраняются.

6. Безопасность

Мы принимаем разумные меры для защиты данных, включая использование защищённых соединений (HTTPS).

7. Дети

Приложение предназначено для использования педагогами и не собирает данные детей.

8. Контакты

По вопросам конфиденциальности обращайтесь: info@robospark.kz

English

1. General

The "RoboShare Projector" application (the "App") is developed by RoboSpark. This Privacy Policy describes what data is collected and how it is used.

2. Data We Collect

  • Network information — Wi-Fi network name to connect to the server.
  • Bluetooth data — to receive presentations from the teacher's device.
  • Location data — solely to determine the Wi-Fi network name (Android 10+ requirement). Location data is not stored or transmitted.

3. How We Use Data

Collected data is used exclusively for:

  • Connecting to the RoboSpark server to retrieve schedules and educational materials.
  • Receiving and displaying presentations via Bluetooth.

4. Third-Party Sharing

We do not share personal data with third parties. Data is transmitted only to the RoboSpark server for the App to function.

5. Data Storage

The App stores a cache of educational materials locally on the device. No personal user data is stored.

6. Security

We take reasonable measures to protect data, including the use of secure connections (HTTPS).

7. Children

The App is intended for use by educators and does not collect data from children.

8. Contact

For privacy inquiries, contact: info@robospark.kz
